## Fundamentals
To execute ruby code from the console

`ruby <filename>`
___

To print to the console, 3 options
* puts 
* p
* print

```rb
# 3 ways to print to the console
puts "Hello World"
# p returns whatever arguement you give it
p "Hello World"
# print does not have a newline at the end
print "Hello World"
```
